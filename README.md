# Схема структуры управления ограничениями доступа
1. `admin` - Администратор
   - Полный доступ ко всем ресурсам системы
   - Управление ролями пользователей
   - Просмотр и управление всеми пользователями
   - Деактивация любых пользователей (кроме суперпользователя)
   - Наследует все права модератора и пользователя

2. `moderator` - Модератор
   - Просмотр списка всех пользователей
   - Просмотр детальной информации о пользователях
   - Наследует все права обычного пользователя

3. `user` - Обычный пользователь
   - Управление собственным профилем
   - Изменение своего пароля
   - Мягкое удаление своего аккаунта
